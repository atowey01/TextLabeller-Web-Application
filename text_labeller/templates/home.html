TextLabeller is a free platform for labelling data for text classification tasks. Follow the steps below to start labelling 😊

1. Register as a user or login

2. Create a project which includes uploading a CSV file containing one column of text data and providing the classes you want to label your data into

3. Go to the label data tab and select your project

4. Start labelling!

5. View and download your labelled data   🎉 🎉 {% endblock content %}
